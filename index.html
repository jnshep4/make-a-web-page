Section 0 Notes

Internet- Browser sends HTTP request to server, retrieves page. Internet is connection of HTML pages with all internet-connected computers and devices.

HTML-Hypertext Markup Language. language that makes up most content on the web. HTML contains:

Text content - what you see

Markup - how it looks

Tags and Elements-HTML documents are made up of elements. Browsers read the type of each element using HTML tags (i.e. < b >, < p >, etc.).

Stupid- Computers only do exactly what you tell them. Think evil genie.

Inline v. Block- Inline elements flow with text content, while Block elements create a "box" around the content they are referencing (single unit?)
